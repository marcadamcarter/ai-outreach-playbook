# AI Outreach Playbook

A modular system for local AI safety awareness outreach. Seven deployment modes — from placing a flyer in a coffee shop to briefing a city council — each with its own playbook, effort level, and sponsor posture.

## The 7 Modes

| Mode | Effort | Sponsor needed? |
|------|--------|-----------------|
| [Static Presence]({{ "/playbooks/static-presence/" | relative_url }}) | Very low | Optional |
| [Signal Actions]({{ "/playbooks/signal-actions/" | relative_url }}) | Very low | Optional |
| [Interactive Booths]({{ "/playbooks/interactive-booths/" | relative_url }}) | Moderate | Supported |
| [Micro-Gatherings]({{ "/playbooks/micro-gatherings/" | relative_url }}) | Low-moderate | Optional |
| [Digital Amplification]({{ "/playbooks/digital-amplification/" | relative_url }}) | Async | Supported |
| [Individual Advocacy]({{ "/playbooks/individual-advocacy/" | relative_url }}) | Zero | Not required |
| [Institutional Touchpoints]({{ "/playbooks/institutional-touchpoints/" | relative_url }}) | Variable | Optional |
