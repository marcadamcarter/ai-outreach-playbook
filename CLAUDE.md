This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Modular toolkit for local AI risk awareness outreach (GuardRailNow). Seven deployment modes — from placing flyers to briefing city councils — each with operational playbooks, effort levels, and sponsor guidance.

- **Public site**: https://marcadamcarter.github.io/ai-outreach-playbook/
- **License**: Content CC BY 4.0, Code MIT

## Repository Layout

- **`playbooks/`** — One operational guide per mode (7 files)
- **`metrics/`** — After-Action Report template
- **`docs/`** — Public-facing Jekyll site (mirrors key files)
- **`core-model.md`** — Strategic rationale for all 7 modes
- **`START-HERE.md`** — Entry point for new readers
- **`RECRUITING.md`** — Volunteer onboarding
- **`sponsoring.md`** — Sponsor guide
- **`pitch.md`** — One-page project pitch

## Communication Standard

All assets follow the **3-Layer Message Model**: (1) Hook: 2-5 words, (2) Core idea: one sentence, (3) Action: QR code or next step. Assets with a fourth element should be rejected.
